# Power BI Superstore Project

This repository contains a Power BI dashboard project analyzing Superstore sales data.

## Project Structure

- `PBIX/` – Power BI Desktop report files (.pbix)
- `Data/` – Sample or anonymized datasets
- `Scripts/` – Data preparation or analysis scripts
- `Docs/` – Documentation and screenshots

## How to Open

1. Download or clone this repository.
2. Open the `.pbix` file in Power BI Desktop.
3. Connect to data in the `Data/` folder if needed.

## Features

- Sales by region, category, and segment
- Top products analysis
- Trend analysis with time series visuals
- Interactive slicers and filters

## Notes

- This is a demo project; no sensitive data is included.
- Make sure you have Power BI Desktop installed.
